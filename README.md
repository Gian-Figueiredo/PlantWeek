# PlantWeek

PlantWeek é para horários semanais, eventos, tarefas, ou seja, um planejador semanal

## Funcionalidades

### Calendário Semanal
- Planeje sua semana com um Calendário Semanal, que além de organizar suas tarefas recorrentes, também calcula quanto tempo é gasto em determinada atividade durante a semana, por meio de um relatório personalizado por meio de tags criadas por você

### Eventos
- Calendários cheio de eventos é massivo e difícil de ler, agora você pode pesquisar se tal horário estar disponível e, caso não estiver, é mostrado a melhor recomendação, ou se quiser sobreescrever a anterior também é possível

### Tarefas
- Todo dia você precisa realizar tarefas, sejam elas as mesmas ou não, agora fica bem mais fácil gerenciá-las. Crie, complete e termine seus afazeres sem perda de foco ou de tempo
